## Notes

### Programming Types
* Declaritive **
* Imperative

### Create React Project

`$ npm create vite@latest` Choices: {.|project-name}, React, JavaScript
`cd {project-name}` if given a name then cd into directory
`$ npm {i|install}`

### Set Port

In `vite.config.js`:

`export default defineConfig({`
`    ***`
`    server: {`
`        host: "0.0.0.0",`
`        port: 3000,`
`    }`
`    ***`
`});`

### Run React Project

`$ npm run dev`
